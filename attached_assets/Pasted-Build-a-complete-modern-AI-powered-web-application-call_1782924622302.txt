Build a complete, modern, AI-powered web application called "Smart Autonomous Shopping AI Agent".

This should NOT be a normal shopping website and NOT be a simple chatbot.

The core idea is that the AI Agent performs all decision making automatically with minimal user effort.

The application should look like a premium AI product with a modern responsive UI.

======================================================
PROJECT GOAL
======================================================

Create an autonomous AI Shopping Assistant that behaves like a personal shopping expert.

Instead of simply searching products, the AI should:

• Understand user intent.
• Analyze user preferences.
• Compare products.
• Rank products.
• Explain recommendations.
• Predict user needs.
• Suggest alternatives.
• Learn from conversation context.
• Automate shopping decisions.

The AI should perform reasoning before responding.

Every recommendation should include an explanation.
